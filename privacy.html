Permissions

The extension`s manifest has these permissions :

  • storage : To cache the fetched/searched process groups.

  • activeTab : To get the details about the currently active tab, to check if its a logged in Nifi instance or not.

  • tabs : To get the tab details like tab id, url.

  • contextMenus : To add a right-click menu shortcut to recently opened Nifi environments through extension.

  • identity : To enable the Oauth access to the user`s logged in google account. So user can upload any flow as template to their own google drive using a custom UI component.

  • identity.email : To display the logged in user`s email inside the extension.

  • *://*/nifi/* : To inject content scripts on url`s matching a Nifi instance`s url pattern.

  • unlimitedStorage : To store templates data for copy flow and paste flow functionality.
